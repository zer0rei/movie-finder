# [Movie Finder](http://zer0rei.github.io/movie-finder/)

Browse [Movies and TV Series](http://zer0rei.github.io/movie-finder/).

## Setup

### Install
```sh
yarn install
yarn start
```

### Test
```sh
yarn test
```

### Deploy to gh-pages
```sh
yarn deploy
```
